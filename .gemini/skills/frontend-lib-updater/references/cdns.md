# CDN URL Patterns

## jsDelivr (Preferred)
- **Base**: `https://cdn.jsdelivr.net/npm/`
- **Pattern**: `[package]@[version]/dist/[file]`
- **Examples**:
    - `jquery-validation@1.21.0/dist/jquery.validate.min.js`
    - `bootstrap@5.3.3/dist/css/bootstrap.min.css`

## cdnjs
- **Base**: `https://cdnjs.cloudflare.com/ajax/libs/`
- **Pattern**: `[package]/[version]/[file]`
- **Examples**:
    - `jquery-validation/1.21.0/jquery.validate.min.js`
    - `jquery/3.7.1/jquery.min.js`
